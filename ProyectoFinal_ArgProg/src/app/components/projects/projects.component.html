Encriptador de texto

Un simple encriptador (y desencriptador) terminado el 04/05/2022. Escribir cualquier texto y apretar el botón de encriptar. Para comprobar que funciona bien, apretar el botón de copiar texto y luego intentar desencriptarlo. Construido con HTML, CSS y JS. Clickear imagen para dirigirse al proyecto.
